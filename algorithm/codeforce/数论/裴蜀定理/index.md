

裴蜀定理（Bézout's identity），也叫裴蜀等式，是数论中一个关于最大公约数 (Greatest Common Divisor, GCD) 的重要定理。它得名于法国数学家艾蒂安·裴蜀（Étienne Bézout）。📘 裴蜀定理的陈述对于任意两个整数 $a$ 和 $b$，设 $d$ 是它们的最大公约数，即 $d = \gcd(a, b)$。核心结论关于未知数 $x$ 和 $y$ 的线性丢番图方程（Linear Diophantine Equation）：$$ax + by = c$$有整数解 $(x, y)$ 的充要条件是 $c$ 必须是 $a$ 和 $b$ 的最大公约数 $d$ 的倍数。特别重要的推论 (裴蜀等式)当 $c$ 等于最大公约数 $d$ 时，方程 $ax + by = d$ 必然存在整数解 $x$ 和 $y$。也就是说，对于任意两个整数 $a$ 和 $b$，总存在整数 $x$ 和 $y$，使得：$$ax + by = \gcd(a, b)$$



 裴蜀定理的用途和意义1. 判断方程可解性裴蜀定理最直接的作用是，它提供了一个判断形如 $ax + by = c$ 的方程是否有整数解的简单方法：有解 $\iff c \pmod{\gcd(a, b)} = 0$2. 互质的条件如果 $a$ 和 $b$ 互质（即 $\gcd(a, b) = 1$），则方程 $ax + by = 1$ 一定有整数解。这是判断两个数互质的一个非常方便的代数条件。3. 构造性求法这些整数解 $(x, y)$ 称为裴蜀数。虽然裴蜀定理保证了解的存在性，但要具体求出这些解，通常需要使用扩展欧几里得算法 (Extended Euclidean Algorithm)。4. 推广到多个整数裴蜀定理可以推广到 $n$ 个整数 $a_1, a_2, \dots, a_n$：设 $d = \gcd(a_1, a_2, \dots, a_n)$，那么一定存在整数 $x_1, x_2, \dots, x_n$，使得：$$a_1 x_1 + a_2 x_2 + \dots + a_n x_n = d$$
