简单来说，**FWT (Fast Walsh-Hadamard Transform，快速沃尔什-阿达马变换)** 是针对位运算（AND, OR, XOR）的一种特殊的生成函数工具。

如果你熟悉 **FFT (快速傅里叶变换)**，FFT 是用来加速**加法卷积**的（即多项式乘法）；而 FWT 是用来加速**位运算卷积**的。

在博弈论和组合数学中，我们经常遇到这种问题：“从集合 A 中选一个数 $a$，从集合 B 中选一个数 $b$，使得 $a \oplus b = k$ 的方案数是多少？”这就是一个典型的 **XOR 卷积**问题。

---

### 1. 为什么需要 FWT？

假设有两个频率数组 $A$ 和 $B$：

* $A[i]$ 表示 SG 值为 $i$ 的怪物数量。
* $B[j]$ 表示 SG 值为 $j$ 的怪物数量。

我们要找有多少对 $(i, j)$ 满足 $i \oplus j = 0$。普通的暴力做法是两层循环：

```python
for i in range(N):
    for j in range(N):
        C[i ^ j] += A[i] * B[j]

```

复杂度是 $O(N^2)$。当 $N = 5 \times 10^5$ 时，这根本跑不动。

**FWT 的作用**就是把这个过程优化到 $O(N \log N)$。它的逻辑和 FFT 非常像：

1. **转换：** 将数组 $A$ 和 $B$ 通过 FWT 变成“点值形态” $\hat{A}$ 和 $\hat{B}$。
2. **乘法：** 在点值形态下，直接做按位相乘：$\hat{C}[i] = \hat{A}[i] \times \hat{B}[i]$。
3. **逆转换：** 将结果 $\hat{C}$ 通过 **IFWT** 变回普通的数组 $C$。

---

### 2. XOR 卷积的原理

对于 XOR（异或）运算，FWT 利用了一个非常优美的数学性质：


$$FWT(A \oplus B) = FWT(A) \cdot FWT(B)$$

对于两个长度为 $2$ 的数组 $[a_0, a_1]$，它的 FWT 转换公式是：

* $FWT(A) = [a_0 + a_1, a_0 - a_1]$

对于更长的数组，它通过分治（类似递归）的方法，每一步都进行这种“加减”操作。

---

### 3. 在本题中如何使用？

在本题中，你有 $R$ 个怪物，每个怪物的 SG 值分布是一样的。你想求 $R$ 个 SG 值异或起来等于 0 的方案数。

这本质上是把同一个数组 $A$ 做了 $R$ 次卷积：

1. 计算 $A$ 的 FWT 变换：$\hat{A} = FWT(A)$。
2. 因为要做 $R$ 次，所以在点值形态下就是 $R$ 次方：$\hat{Res}[i] = (\hat{A}[i])^R$。
* *注意：这里需要用到**快速幂**来计算 $(\hat{A}[i])^R \pmod{Mod}$。*


3. 最后做逆变换：$Res = IFWT(\hat{Res})$。
4. 最终的 $Res[0]$ 就是你要的“异或和为 0”的总方案数。

---

### 4. 总结对比

| 变换类型 | 解决的问题 (卷积类型) | 核心数学操作 |
| --- | --- | --- |
| **FFT** | 加法卷积 ($a + b = k$) | 复数单位根、多项式乘法 |
| **FWT (OR)** | 或运算卷积 ($a \mid b = k$) | 前缀和思想 |
| **FWT (AND)** | 与运算卷积 ($a \& b = k$) | 后缀和思想 |
| **FWT (XOR)** | **异或运算卷积 ($a \oplus b = k$)** | **蝴蝶操作 (加减运算)** |

**一句话总结：**
FWT 就是一种让我们能像做普通乘法一样去处理“异或组合问题”的数学魔术。它能把 $R=10^9$ 这种天文数字级别的组合，通过快速幂和对数级的变换轻松算出来。
