# 线性回归优化 与2维？偏导方程组求解
岭回归（Ridge Regression）
克莱姆法则（Cramer's Rule）
https://codeforces.com/gym/105446/problem/B
https://github.com/Yawn-Sean/Daily_CF_Problems/blob/main/daily_problems/2026/04/0420/solution/cf105446b.md

你的分析非常精准。这道题的本质是将**岭回归（Ridge Regression）**的解析解与**前缀和（Prefix Sum）**优化结合起来。

以下是完整的逻辑推导和处理流程：

### 1. 岭回归的解析解推导

我们要最小化罚函数：
$$p(K,B)=\sum_{i=L}^R (Kx_i+B-y_i)^2 + \lambda(K^2+B^2)$$

分别对 $B$ 和 $K$ 求偏导并令其为 $0$：

* **对 $B$ 求偏导：**
    $$\frac{\partial p}{\partial B} = 2 \sum (Kx_i + B - y_i) + 2\lambda B = 0$$
    $$K \sum x_i + B(n' + \lambda) = \sum y_i$$
    （其中 $n' = R - L + 1$ 是区间内的月份数）

* **对 $K$ 求偏导：**
    $$\frac{\partial p}{\partial K} = 2 \sum (Kx_i + B - y_i)x_i + 2\lambda K = 0$$
    $$K(\sum x_i^2 + \lambda) + B \sum x_i = \sum x_iy_i$$

### 2. 联立方程组

设区间内的各项和为：
* $S_x = \sum x_i$
* $S_y = \sum y_i$
* $S_{xx} = \sum x_i^2$
* $S_{xy} = \sum x_iy_i$

方程组可以写为：
$$\begin{cases} (n' + \lambda)B + S_x K = S_y \\ S_x B + (S_{xx} + \lambda)K = S_{xy} \end{cases}$$

利用克莱姆法则（Cramer's Rule）求解：
* 行列式 $D = (n' + \lambda)(S_{xx} + \lambda) - S_x^2$
* $B = \frac{S_y(S_{xx} + \lambda) - S_{xy} S_x}{D}$
* $K = \frac{(n' + \lambda)S_{xy} - S_y S_x}{D}$



### 3. 算法实现流程

#### 预处理阶段 ($O(n)$)
为了在 $O(1)$ 时间内获取任意区间的 $S_x, S_y, S_{xx}, S_{xy}$，我们需要预处理四个前缀和数组：
1.  `preX[i]` = $\sum_{j=1}^i x_j$
2.  `preY[i]` = $\sum_{j=1}^i y_j$
3.  `preXX[i]` = $\sum_{j=1}^i x_j^2$
4.  `preXY[i]` = $\sum_{j=1}^i x_j y_j$

#### 查询阶段 ($O(m)$)
对于每个查询 $(L, R, \lambda, X)$：
1.  通过前缀和计算 $S_x, S_y, S_{xx}, S_{xy}$ 和 $n' = R - L + 1$。
2.  代入上述公式计算 $K$ 和 $B$。
3.  计算预测值：$Y_{pred} = K \cdot X + B$。

### 4. 关键点提示
* **精度问题**：由于涉及平方和及乘积和，数值可能会非常大，建议使用 `double` 或 `long double` 进行计算。
* **$\lambda$ 的作用**：当 $\lambda > 0$ 时，行列式 $D$ 永远大于 $0$（因为这是一个正定矩阵的行列式），这保证了方程组一定有唯一解，同时也防止了过拟合。

这种方法将原本需要 $O(n)$ 遍历的最优化问题，通过前缀和技巧降到了 $O(1)$ 的查询复杂度，非常适合处理大规模的区间回归预测。
