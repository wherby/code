# 题目
https://codeforces.com/gym/106414/problem/K
https://github.com/Yawn-Sean/Daily_CF_Problems/blob/main/daily_problems/2026/05/0507/solution/cf106414k.md

这道题的核心在于利用网格图（Grid Graph）的二分图（Bipartite）性质，并在此基础上通过“局部微调”来满足三个特殊格子的约束。

按照你提供的思路，我们可以将解法细化为以下步骤：

### 1. 初始染色（二分染色）

首先，我们将网格进行常规的棋盘式染色，只使用颜色 **A** 和 **B**：

* 若 $(i + j)$ 为偶数，染颜色 **A**。
* 若 $(i + j)$ 为奇数，染颜色 **B**。

此时，任何相邻格子颜色都不同。

---

### 2. 分类讨论特殊格子 $S_1, S_2, S_3$

由于初始只用了 A 和 B 两种颜色，根据抽屉原理，这三个特殊格子中至少有两个颜色相同。

#### 情况 A：三个格子原本就有两种颜色

（例如：$S_1$ 为 A，$S_2$ 为 A，$S_3$ 为 B）
这种情况下，我们只需要将 $S_1$ 或 $S_2$ 其中一个改为**颜色 C**。

* **动作**：直接把 $S_1$ 的颜色改为 C。
* **合法性**：由于 C 是全新的颜色，它绝不会和 $S_1$ 原有的相邻格子（颜色 B）冲突。此时三个特殊格子颜色分别为 C, A, B，满足条件。

#### 情况 B：三个格子原本颜色全部相同

（例如：$S_1, S_2, S_3$ 均为颜色 A）
这意味着这三个格子的坐标之和 $(i+j)$ 的奇偶性相同。我们要把它们改为不同的颜色（A, B, C）。

* **步骤 1**：将 $S_1$ 改为 **颜色 C**。这不会引起冲突。
* **步骤 2**：尝试将 $S_2$ 改为 **颜色 B**。
* **冲突检查**：如果 $S_2$ 改为 B 后，与其相邻的格子中没有原本就是 B 的，则成功。但在二分染色中，$S_2$（原为 A）的所有邻居**一定是 B**。
* **解决办法（微调）**：要让 $S_2$ 变成 B，必须同时改变 $S_2$ 邻居的颜色。



---

### 3. 微调策略与无解判定

正如提示 2 所言，当三个格子同色时，我们需要通过局部路径的颜色翻转来调整。

#### 寻找调整路径

我们要将 $S_2$ 从 A 变为 B。这要求 $S_2$ 的所有邻居从 B 变为 C。

* 如果 $S_2$ 的所有邻居在变色为 C 后，不与 $S_1$（已经是 C）冲突，则调整成功。
* **关键点**：只要 $S_2$ 与 $S_1$ 的距离足够远（曼哈顿距离 $dist(S_1, S_2) \ge 2$），我们总能找到一种局部调整方案。

#### 无解情况的判定

唯一可能导致无解的情况是：**网格结构极端受限，使得我们无法在不违反相邻原则的前提下，给三个特殊格子分配三种颜色。**

在标准的 $n \times m$ 网格中，只有当 $n, m$ 极小时（例如 $1 \times 3$ 或某些极窄区域），且三个特殊格子之间的位置关系导致了色彩强制性闭环（例如：三个格子两两之间距离都非常近，且共享了所有的邻域空间）。

**总结：**

1. **二分染色** A 和 B。
2. 若特殊格色占两种，改一个为 C。
3. 若同色，选一个改 C，选另一个通过反转其邻域颜色来改 B。
4. 若无法反转（通常见于极小规模网格且特殊点距离极近），则无解。

在绝大多数 $n, m \ge 3$ 的情况下，通过这种局部微调，**解是一定存在的**。
