
## Karp 最小平均权值回路算法详解. 
[最大环算法则是相反](../../codeforce/图论/Karp/最大平均环算法.py)

这是理查德·卡普在1978年提出的经典算法，用于在**有向图**中寻找**平均权值最小的回路**（环）。

---

## 一、问题定义

### 基本概念
给定一个有向图 `G = (V, E)`，每条边 `(u, v)` 有权重 `w(u, v)`（可为正、负或零）。

**目标**：找到一个回路 `C`（起点=终点的路径），使得：
```
最小平均权值 = min_{C} (∑_{e∈C} w(e)) / |C|
```
其中 `|C|` 是回路中的边数。

### 实际应用场景
1. **金融套利**：在外汇交易中寻找套利机会（负权回路）
2. **系统性能优化**：找到平均延迟最小的循环路径
3. **调度问题**：优化循环生产线的效率

---

## 二、算法核心思想

### 动态规划状态定义

**关键技巧**：固定一个起点 `s`，定义：
```
dp[k][v] = 从起点 s 出发，恰好经过 k 条边到达顶点 v 的**最短路径长度**
```

### 状态转移方程

```
初始化：
dp[0][s] = 0
dp[0][v] = ∞ (v ≠ s)

递推：
for k = 1 to n:
    for each edge (u, v) ∈ E:
        dp[k][v] = min(dp[k][v], dp[k-1][u] + w(u, v))
```

这实际上就是 **Bellman-Ford 算法的扩展**，但记录的是恰好 k 步的最短路径。

---

## 三、核心公式

Karp 证明了最小平均权值回路 `λ*` 可以通过以下公式计算：

```
λ* = min_{v ∈ V} max_{0 ≤ k ≤ n-1} [(dp[n][v] - dp[k][v]) / (n - k)]
```

### 公式解读

1. **`dp[n][v]`**：从 s 出发，用 n 步到达 v 的最短路径
2. **`dp[k][v]`**：从 s 出发，用 k 步（k < n）到达 v 的最短路径
3. **`(dp[n][v] - dp[k][v])`**：这 n-k 步形成的回路的**总权重**
4. **`/(n - k)`**：除以步数，得到该回路的**平均权重**
5. **`max_{0 ≤ k ≤ n-1}`**：对于固定 v，取最大值（最坏情况）
6. **`min_{v ∈ V}`**：在所有顶点中取最小值（最优回路）

## 四、完整算法步骤

```python
def karp_minimum_mean_cycle(graph, n, edges):
    """
    graph: 有向图
    n: 顶点数（假设顶点编号 0 到 n-1）
    edges: 边列表 [(u, v, w), ...]
    """
    INF = float('inf')
    
    # 1. 初始化 DP 表：dp[k][v]
    dp = [[INF] * n for _ in range(n + 1)]
    dp[0][0] = 0  # 选择顶点 0 作为起点
    
    # 2. 动态规划填表
    for k in range(1, n + 1):
        for u, v, w in edges:
            if dp[k-1][u] != INF:
                dp[k][v] = min(dp[k][v], dp[k-1][u] + w)
    
    # 3. 计算最小平均权值
    min_mean = INF
    for v in range(n):
        # 跳过无法从起点到达的顶点
        if dp[n][v] == INF:
            continue
        
        # 计算该顶点的最大平均权值
        max_mean_for_v = -INF
        for k in range(n):
            if dp[k][v] != INF:
                mean = (dp[n][v] - dp[k][v]) / (n - k)
                max_mean_for_v = max(max_mean_for_v, mean)
        
        min_mean = min(min_mean, max_mean_for_v)
    
    return min_mean
```

### 时间复杂度
- **O(VE)**：V 个顶点，E 条边
- 两层循环：外层 k=1..n，内层遍历所有边

### 空间复杂度
- **O(V²)**：可以优化到 O(V) 只保留两行

## 五、手动计算示例

### 示例图
```
顶点：0, 1, 2
边：
0 → 1 (权重 5)
1 → 2 (权重 -3)
2 → 0 (权重 -2)
2 → 1 (权重 1)

图示：
0 --(5)--> 1 --(-3)--> 2
^                    |
|____(-2)____________|
```

### 可能回路
1. **0→1→2→0**：总权值 = 5 + (-3) + (-2) = 0，边数 = 3，平均值 = 0
2. **1→2→1**：总权值 = (-3) + 1 = -2，边数 = 2，平均值 = -1

**最小平均值** = -1（回路 1→2→1）

### 算法计算过程

选择起点 s = 0

**步骤1：DP 填表**

| k\v | 0 | 1 | 2 |
|-----|---|---|---|
| 0 | 0 | ∞ | ∞ |
| 1 | ∞ | 5 | ∞ |
| 2 | ∞ | min(5+?=?)  | 5+(-3)=2 |
| 3 | -2 | ... | ... |

**步骤2：应用公式**

对于 v=2, k=1：`(dp[3][2] - dp[1][2]) / (3-1)` = (某值)

最终算法会正确识别出平均值 -1 的回路。

## 六、算法原理证明（简要）

### 关键引理
对于任意回路 C，其平均权值 λ 满足：
- 如果从起点 s 经过若干步进入回路，然后在回路中绕圈，路径长度的增长是线性的
- `dp[k][v]` 随着 k 增加，其行为由回路平均权值主导

### 直观理解
- 当 k 足够大时，`dp[k][v] ≈ k·λ + c(v)`
- `dp[n][v] - dp[k][v] ≈ (n-k)·λ`
- 通过比较不同步数，可以消除起点选择的影响

## 七、优化与变体

### 1. 空间优化
只保留两行 DP 表：
```python
dp_prev = [INF] * n
dp_curr = [INF] * n
```

### 2. 检测负权回路
如果 `λ* < 0`，说明存在平均权值为负的回路（可用于套利检测）

### 3. 找回路本身
需要记录前驱节点，最后回溯重建路径

### 4. 与二分查找结合
可以用二分查找 + 边权调整来近似求解

## 九、实际应用代码示例

### 金融套利检测
```python
def detect_arbitrage(exchange_rates):
    """
    exchange_rates: 货币对汇率矩阵
    返回是否存在套利机会
    """
    n = len(exchange_rates)
    # 转换：取负对数，套利变成负权回路
    edges = []
    for i in range(n):
        for j in range(n):
            if i != j:
                weight = -math.log(exchange_rates[i][j])
                edges.append((i, j, weight))
    
    # 使用 Karp 算法
    min_mean = karp_minimum_mean_cycle(n, edges)
    
    # 如果最小平均权值为负，存在套利
    return min_mean < 0
```

## 总结

**Karp 最小平均权值回路算法**的精髓在于：
1. 用 **动态规划** 记录恰好 k 步的最短路径
2. 通过 **比较不同步数的路径差异** 来推断回路性质
3. 公式 `min max (dp[n][v] - dp[k][v])/(n-k)` 巧妙地分离了回路与路径

这是一个优雅且高效的算法，是图论和算法设计中的经典之作。
