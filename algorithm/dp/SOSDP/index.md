
# 求集合数量 SOS 
[GCD数组数目](../../mathA/gcd/数论GCD)
OR数组数目




莫比乌斯反演 是对 整除代数结构 上的容斥原理的推广；而 SOS DP 则是对 布尔代数结构 (子集关系) 上容斥原理的推广。

=============

|特性 | GCD 计数问题 (莫比乌斯)|位运算计数问题 (SOS DP)|
| :------- | :------: | -------: |
|关系 | 整除关系 |($gk \cdot g$)|
|聚合目标|恰好为 $g$ 的子序列数量 ($\mathbf{G}[g]$)|恰好满足 某种位条件的子集数量|
|莫比乌斯反演|是必要的。用于将“至少”（倍数）关系反演为“恰好”（GCD）关系。|不是必需的。SOS DP 直接处理集合间的包含关系。|
