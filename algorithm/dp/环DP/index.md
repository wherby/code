为什么要枚举两次？在一个环 $r_0 \to r_1 \to \dots \to r_{k-1} \to r_0$ 中，为了满足最大独立集，$r_0$ 和 $r_{k-1}$ 至少要死一个（或者都死）。第一次 DP 覆盖了“$r_0$ 必死，$r_{k-1}$ 随意”的情况。第二次 DP 覆盖了“$r_{k-1}$ 必死，$r_0$ 随意”的情况。两者取最大值，就完美涵盖了所有合法的存活方案。
