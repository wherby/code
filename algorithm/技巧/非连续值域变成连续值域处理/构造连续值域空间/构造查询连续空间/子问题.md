## 区间标记查询的问题
已知N个区间标记[li,ri]表示区间被选中，求区间[lq,rq] 被多少个区间标记包含？

使用下面的代码就可以把区间包含的问题。变成连续值域的区间标记问题。
使用第一次循环，把[l,r] 的标记 扩展到[l][r-x] 行标记
使用第二次循环，把[l][r-x1]的标记，扩展到[l-x2][r-x1]，这样查询任何[l-x2][r-x1] 都能得到[l][r]的贡献， 但是标记扩散的时候 可能出现 l-x2> r-x1， 这个区间是没有意义的，而且不会有这个值域的查询，所以没问题 
```python 
    for _ in range(q):
        l, r = GMI()
        cnt[l][r] += 1
    
    for l in range(n):
        for r in range(n - 1, 0, -1):
            cnt[l][r - 1] += cnt[l][r]
    
    for l in range(1, n):
        for r in range(n):
            cnt[l][r] += cnt[l - 1][r]
```
