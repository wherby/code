你说的是**空间填充曲线（Space-filling curve）**！特别是：

## **Z-order curve（莫顿编码，Morton code）**

这是一种将二维空间映射到一维的方法，使得**空间上相邻的点在一维索引上也尽可能相邻**。

---

## 核心原理

### 莫顿编码（Morton Code）
通过**交错二进制位**将二维坐标映射到一维整数：

```python
def morton_encode(x, y):
    """将二维坐标 (x, y) 编码为一维莫顿码"""
    # 将 x 和 y 的二进制位交错排列
    result = 0
    for i in range(32):  # 假设坐标在 32 位范围内
        result |= ((x >> i) & 1) << (2*i)
        result |= ((y >> i) & 1) << (2*i + 1)
    return result

def morton_decode(z):
    """从莫顿码解码出二维坐标"""
    x = y = 0
    for i in range(32):
        x |= ((z >> (2*i)) & 1) << i
        y |= ((z >> (2*i + 1)) & 1) << i
    return x, y
```

### 交错过程示例
```
x = 3 (二进制: 11)
y = 2 (二进制: 10)

x 的位:  1 1
y 的位:  1 0
交错后: 1 1 1 0
结果: 二进制 1110 = 14
```

## 其他空间填充曲线

### 1. **希尔伯特曲线（Hilbert Curve）**
- 更好的局部性保持
- 相邻点在曲线上更接近
- 常用于数据库索引（如 PostGIS）

```python
def hilbert_index(x, y, pow=4):
    """计算希尔伯特曲线索引（pow 是阶数）"""
    # 实现较复杂，需要通过旋转和反射
    # 这里给出简化版本
    index = 0
    for s in range(pow-1, -1, -1):
        rx = (x >> s) & 1
        ry = (y >> s) & 1
        if rx == 0 and ry == 0:
            # 不旋转
            pass
        elif rx == 0 and ry == 1:
            # 旋转 90 度
            x, y = y, x
        elif rx == 1 and ry == 0:
            # 旋转 270 度
            x, y = ~y, ~x
        else:
            # 旋转 180 度
            x, y = ~x, ~y
        index = (index << 2) | (rx * 2 + ry)
    return index
```

### 2. **皮亚诺曲线（Peano Curve）**
- 三维空间填充
- 更复杂，但局部性更好

### 3. **格雷码（Gray Code）**
- 相邻数字只有一位不同
- 可用于一维索引优化

### 1. **四叉树/八叉树索引**
```python
class QuadTree:

        # 使用莫顿码作为键值
```

### 2. **图片/地图分块存储**
```python
def get_tile_index(zoom, x, y):
    """Google Maps 风格的瓦片索引"""
    # 使用莫顿码将 (x,y) 映射为一维索引
    return morton_encode(x, y)
```

### 3. **游戏地图的存储和寻路**
```python
# 将二维地图展平为一维，保留空间局部性
def map_to_1d(x, y, width):
    # 使用莫顿码而不是简单的 y*width + x
    return morton_encode(x, y)

# 获取相邻位置的索引（虽然不是完全相邻，但比行主序好）
def get_neighbor_indices(x, y):
    neighbors = []
    for dx, dy in [(-1,0), (1,0), (0,-1), (0,1)]:
        nx, ny = x + dx, y + dy
        neighbors.append(morton_encode(nx, ny))
    return neighbors
```

## 性能对比

假设一个 8×8 的地图：

| 方法 | 相邻点在一维中的平均距离 | 局部性 |
|------|------------------------|--------|
| 行主序 (y*width + x) | 较大（跨行时跳跃） | 差 |
| 列主序 | 同左 | 差 |
| **莫顿编码** | 较小 | 好 |
| **希尔伯特曲线** | 最小 | 最好 |

## 完整的莫顿编码实现

```python
class MortonIndex:
    def __init__(self, width, height):
        self.width = width
        self.height = height
        # 预计算编码表加速
        self.encode_lut = [self._encode_byte(b) for b in range(256)]
    
    def _encode_byte(self, b):
        """编码一个字节（8位）"""
        result = 0
        for i in range(8):
            if (b >> i) & 1:
                result |= 1 << (2*i)
        return result
    
    def encode(self, x, y):
        """快速莫顿编码"""
        # 分字节处理
        result = 0
        for i in range(4):  # 假设 32 位坐标
            byte_x = (x >> (8*i)) & 0xFF
            byte_y = (y >> (8*i)) & 0xFF
            encoded = self.encode_lut[byte_x] | (self.encode_lut[byte_y] << 1)
            result |= encoded << (16*i)
        return result
    
    def get_neighbor_range(self, x, y, radius=1):
        """获取某个区域内的所有索引（用于范围查询）"""
        indices = []
        for dx in range(-radius, radius + 1):
            for dy in range(-radius, radius + 1):
                nx, ny = x + dx, y + dy
                if 0 <= nx < self.width and 0 <= ny < self.height:
                    indices.append(self.encode(nx, ny))
        return sorted(indices)  # 排序后局部性更好
```

## 在算法题中的应用

如果你是在做算法题，很可能需要：

1. **将二维坐标映射到一维**，且保持相邻性
2. **利用这种映射进行二分查找或树状数组操作**

例如：
```python
# 二维前缀和 + 莫顿编码
class Fenwick2D:
    def __init__(self, size):
        self.size = size
        self.bit = [0] * (size * size)
    
    def update(self, x, y, val):
        idx = morton_encode(x, y)
        while idx < len(self.bit):
            self.bit[idx] += val
            idx += idx & -idx
    
    def query(self, x, y):
        # 利用莫顿编码进行二维区间查询
        pass
```

## 选择建议

- **如果只需要简单映射**：用莫顿编码（实现简单）
- **如果需要最好的局部性**：用希尔伯特曲线（实现复杂）
- **如果用于数据库索引**：用希尔伯特曲线（如 PostGIS 的 SP-GiST）
- **如果是竞赛题**：大概率是莫顿编码
