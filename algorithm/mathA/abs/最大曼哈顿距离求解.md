
# 曼哈顿距离的绝对值展开
把N维的距离展开成为 2**N 维的空间，求 D(j) - D(2**N-1-j) 的最大值
https://codeforces.com/gym/106164/problem/G
https://github.com/Yawn-Sean/Daily_CF_Problems/blob/main/daily_problems/2026/05/0515/solution/cf106164g.md


这道题的核心在于将**曼哈顿距离的绝对值拆解**，并利用**树形 DP** 在 $O(2^d \cdot N)$ 的时间内解决。你提供的提示已经点出了最关键的一步：通过枚举符号位来消除绝对值符号。

以下是该问题的详细逻辑梳理与实现思路：

### 1. 核心数学转换

曼哈顿距离的本质是 $2^3 = 8$ 种线性组合的最大值：


$$|x_u - x_v| + |y_u - y_v| + |z_u - z_v| = \max_{s \in \{-1, 1\}^3} \sum_{k=1}^{3} s_k(coord_{u,k} - coord_{v,k})$$

我们将整体目标函数（轨道满意度 + 火箭满意度）展开，设轨道路径权重为 $W(u, v)$：


$$f(u, v) = W(u, v) + (s_1 x_u + s_2 y_u + s_3 z_u) - (s_1 x_v + s_2 y_v + s_3 z_v)$$

为了方便 DP，我们将路径 $W(u, v)$ 拆解为到根节点的距离 $D(u)$。当 $a = \text{LCA}(u, v)$ 时：


$$W(u, v) = D(u) + D(v) - 2D(a)$$

代入后，目标变为极大化：


$$\left[ D(u) + (s_1 x_u + s_2 y_u + s_3 z_u) \right] + \left[ D(v) - (s_1 x_v + s_2 y_v + s_3 z_v) \right] - 2D(a)$$

### 2. 树形 DP 设计

对于每一种符号组合 $s = (s_1, s_2, s_3)$，我们定义每个点的“特征值”：

* **正权点值：** $P_i(s) = D(i) + (s_1 x_i + s_2 y_i + s_3 z_i)$
* **负权点值：** $M_i(s) = D(i) - (s_1 x_i + s_2 y_i + s_3 z_i)$

我们在树上进行一次 DFS，对于每个节点 $a$，维护其子树内：

* $maxP(a, s)$：节点 $a$ 子树中 $P_i(s)$ 的最大值。
* $maxM(a, s)$：节点 $a$ 子树中 $M_i(s)$ 的最大值。

**转移与更新：**
当遍历到节点 $a$ 的子节点 $v$ 时，先利用已有子树信息更新全局答案：

1. $Ans = \max(Ans, maxP(a, s) + maxM(v, s) - 2D(a))$
2. $Ans = \max(Ans, maxM(a, s) + maxP(v, s) - 2D(a))$

然后更新 $a$ 的维护值：

* $maxP(a, s) = \max(maxP(a, s), maxP(v, s))$
* $maxM(a, s) = \max(maxM(a, s), maxM(v, s))$

### 3. 算法流程

1. **预处理：** DFS 一遍求出所有点到根的轨道距离 $D(i)$。
2. **枚举状态：** 循环 $2^3 = 8$ 次，每次对应一种 $(s_1, s_2, s_3)$ 组合。
* 在每次循环中，跑一遍上述的树形 DP。
* 注意：$maxP$ 和 $maxM$ 的初始值应设为该节点自身的 $P_a(s)$ 和 $M_a(s)$，然后再与子树合并。


3. **结果：** 8 次 DP 后的最大值即为所求。

### 4. 复杂度分析

* **时间复杂度：** $O(2^d \cdot N)$。其中 $d=3$（维度），$N$ 为行星数量。对于典型的 $N=10^5$ 量级，这个复杂度非常优秀。
* **空间复杂度：** $O(N)$，主要用于存储树结构和坐标。

**小贴士：**
在实现时，你可以通过位运算 `(mask >> i) & 1` 来快速生成 8 种 `(1, -1)` 的组合，这样代码会非常简洁。
