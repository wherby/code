# Ax +b = 0 mod m 在[0,N] 的解

在这段代码的核心循环中，这一行：

```python
g = gcd(A, m)
if B % g == 0:
    p, q, r = A // g, B // g, m // g
    mn = (-q * inv_mod(p, r)) % r

```

承载了整段代码最核心的**数论推导（同余方程求解）**。

下面为你拆解为什么 `g` 是 `gcd(A, m)`，以及 `p, q, r` 分别代表什么。

---

## 1. 为什么 `g = gcd(A, m)` ？

我们的核心目标是找出有多少个整数 $j$ 使得 $Aj + B$ 是 $m$ 的倍数。
用数学语言描述，就是求以下**一元线性同余方程**在 $j \in [0, N-1]$ 范围内的解的个数：


$$Aj + B \equiv 0 \pmod m \implies Aj \equiv -B \pmod m$$

根据数论中的 **裴蜀定理（Bézout's identity）**，关于 $j$ 的同余方程 $Aj \equiv -B \pmod m$ 有解的**充要条件**是：

> **$-B$ 必须能被 $A$ 和 $m$ 的最大公约数整除。**

因此，代码第一步必须先求出 $A$ 和 $m$ 的最大公约数，即 `g = gcd(A, m)`。
接着，代码用 `if B % g == 0:` 来判断。如果不等于 0，说明这个方程**无解**（即在当前模数 $m$ 下，没有任何一个 $j$ 能让 $Aj+B$ 成为 $m$ 的倍数），直接跳过。

---

## 2. `p, q, r` 分别是什么？

当确定方程有解（`B % g == 0`）后，代码执行了这一步：

```python
p, q, r = A // g, B // g, m // g

```

这实际上是在对同余方程进行**同比例化简（约分）**。
原本的方程是：


$$Aj \equiv -B \pmod m$$

既然 $A, B, m$ 都能被 $g$ 整除，我们把方程的**系数、常数项、以及模数**同时除以 $g$，方程的解不会发生改变。
除以 $g$ 之后的式子变为：


$$\left(\frac{A}{g}\right)j \equiv -\left(\frac{B}{g}\right) \pmod{\frac{m}{g}}$$

对应代码里的变量：

* **`p` ($A/g$)**：化简后的未知数系数。
* **`q` ($B/g$)**：化简后的常数项。
* **`r` ($m/g$)**：**化简后的新模数**。

此时，方程变成了：


$$p \cdot j \equiv -q \pmod r$$

---

## 3. 为什么要费力转化成 `p, q, r`？

**为了能够求逆元（使用 `inv_mod`）。**

在原本的方程中，因为 $A$ 和 $m$ 存在公约数 $g$，使得 $A$ 在模 $m$ 意义下**不存在乘法逆元**（你不能直接把 $A$ 除过去）。

但是，把它们同时除以 $g$ 得到 $p$ 和 $r$ 之后，根据最大公约数的性质，**$p$ 和 $r$ 一定是互质的**（即 $\gcd(p, r) = 1$）。
一旦互质，$p$ 在模 $r$ 的世界里就**必然存在唯一的乘法逆元**了！

代码接下来的操作也就顺理成章了：

```python
mn = (-q * inv_mod(p, r)) % r

```

1. `inv_mod(p, r)` 算出了 $p$ 在模 $r$ 下的逆元 $p^{-1}$。
2. 两边同乘逆元，直接把 $p$ 消掉：$j \equiv -q \cdot p^{-1} \pmod r$。
3. 算出来的 `mn` 就是这个方程在 $[0, r-1]$ 范围内的**最小非负整数解（即第一个满足条件的 $j$）**。

### 举个直观的例子

假设当前 $A=6, B=3, m=9$，我们要解 $6j \equiv -3 \pmod 9$：

1. `g = gcd(6, 9) = 3`。因为 `3 % 3 == 0`，方程有解。
2. 化简：`p = 6//3 = 2`, `q = 3//3 = 1`, `r = 9//3 = 3`。
3. 方程变为：$2j \equiv -1 \pmod 3$。
4. 在模 3 下， $2$ 的逆元是 $2$（因为 $2 \times 2 = 4 \equiv 1 \pmod 3$）。
5. 求解：$j \equiv -1 \times 2 = -2 \equiv 1 \pmod 3$。
6. 得到第一个解 `mn = 1`。之后的解就是 $1, 1+3, 1+6, 1+9 \dots$ 每隔 `r`（也就是 3）出现一次。
