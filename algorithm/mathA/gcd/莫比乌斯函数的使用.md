
# 莫比乌斯函数

把 N个 0,1 的数字任意组合，必然得到的是 2**（n-1) 个奇数和同样数字的偶数。
这里有个性质，把 -C(n,0) + C(n,1)  - C(n,2)+ C(n,3) .... +/- C(n,n) =0
如果把 C(n,0) 去掉就是1， 而组合数的系数符号就是 Mu(x)的计算，所以一个数字形成的影响，等于它的因子的影响和莫比乌斯系数乘积的和
[计算nums的Coprime的cost](计算Coprime数量.py) 这个题目需要计算所有数字在某个点上的coprime 贡献，等于在该点的所有子因数贡献和莫比乌斯系数乘积，这样可以确保每个数字在该点有贡献的话，贡献度为1，不重不漏
等效于把大于或者小于该测试点的数字，映射回了测试点的空间，计算空间与数字分布无关，只与测试点的空间相关
