
# 为了得到 1011111 形状的 低位都是1 的最低位为0 的位置
https://leetcode.cn/problems/construct-the-minimum-bitwise-array-ii/solutions/2948584/o1-ji-suan-mei-ge-shu-pythonjavacgo-by-e-6l9l/
写法一
举例说明：把 101111 取反，得 010000，其 lowbit=10000，右移一位得 1000。把 101111 与 1000 异或，即可得到 100111。

class Solution:
    def minBitwiseArray(self, nums: List[int]) -> List[int]:
        for i, x in enumerate(nums):
            if x == 2:
                nums[i] = -1
            else:
                t = ~x
                nums[i] ^= (t & -t) >> 1
        return nums

写法二
把 101111 加一，得到 110000，再 AND 101111 取反后的值 010000，可以得到方法一中的 lowbit=10000。

class Solution:
    def minBitwiseArray(self, nums: List[int]) -> List[int]:
        for i, x in enumerate(nums):
            if x == 2:
                nums[i] = -1
            else:
                nums[i] ^= ((x + 1) & ~x) >> 1
        return nums
